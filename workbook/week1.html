WEEK 1

HELLO WORLD! [HTML tag!]

In class we participated in group work, displaying our newfound coding knowledge through the correct placement of basic HTML structures. We began this exercise by using the website designercize to create a quick prompt. Our prompt was "a search results view for a live show recommendations app for baristas".

In groups we then worked together to create a basic wireframe for this prompt. Ours featured a grid structure, as well as a title explaining what has been searched, and a filter/sort button.

Finally using HTML tags we worked together to create a basic HTML template for our webpage.

WEEK 1

HUMAN COMPUTER IN YOUR FACE! [gestures]

Through this exercise we explored the significance of gestures in the way we interact with non-humans. This includes gestures such as tapping a card, swiping across a screen or clicking a mouse.

We also explored some gestures that could possibly be used in future Human-Computer Interactions. I explored using the head to scroll, the eyes to click or the hand to hover.
